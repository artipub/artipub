@artipub/cli 是一个artipub cli 工具，方便用户通过命令行进行文章的发布和更新。

## ❓ 为什么需要@artipub/cli?

不想通过js的方式使用@artipub/core，想更加轻量灵活的发布和更新文章至多个平台，写完文章后我就想立马将文章markdown 发布至多个平台，那么此时@artipub/cli 就是为了解决这个问题而衍生出来。
